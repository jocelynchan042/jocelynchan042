## 🎯 Featured Project: World Happiness Levels AI Model
This project explores the use of machine learning models to classify countries into Low, Medium, or High happiness categories based on socio-economic, health, and governance indicators. Using the World Happiness Report dataset, I build and compare Logistic Regression and Random Forest models to evaluate their performance in predicting a country’s happiness class.

## 🎯 Featured Project: Boston Cop Watch AI Chat Platform
This project is a full-stack web application designed to provide accessible, real-time information about Boston police interactions and accountability data. Built with a React frontend and Python backend, the platform integrates the Gemini API to generate contextual AI responses based on structured user-selected scenarios. The system maintains session-based conversational context while delivering a clean, responsive chat experience with dynamic markdown rendering. This project highlights my experience in full-stack development, API integration, and building AI-powered tools focused on public interest and ethical data use.
